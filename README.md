# Basic Ruby
This repo is meant to help people learning ruby the basic principles of the language.

##### Class 1 06/20/15
* Development Environment
* Development tools (text editors, IDE, terminal, irb, pry)
* RVM
* Bundler
* Rubygems
* Git

##### Class 2 06/27/15
* Test Driven Development
* Continuous Integration
* Ruby Documentation
* Ruby sintax and naming conventions
* Ruby variables and constants

##### Class 3 07/04/15
* Numbers
  * Fixnum, Bignum (Ingers)
  * Float, BigDecimal
* Strings
  * ""
  * ''
  * interpolation
  * heredocs
  * concatenation
  * ASCII representation (?A)
* Symbols (DoS and GC)
* Collections
  * Arrays
  * Hashes
  * Ranges
* Flow control
  * if and unless
  * ternary operator
  * elsif and case
  * while and until
* Callable objects
